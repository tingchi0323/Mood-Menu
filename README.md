# Mood Menu 心情選單

一個療癒小網頁，透過點選 emoji 表達今日心情，給予對應的溫暖回應與鼓勵語錄！

## 專案特色

- 根據使用者點選的心情改變背景色與回應文字
- 隨機一句鼓勵語錄
- 可愛互動動畫（emoji 放大、旋轉）
- 清新排版＋柔和配色

## 開發工具

- HTML / CSS / JavaScript
- Git + GitHub Pages
